# Whiteship Live-Study Participant Checker
> 이 프로그램은 해당 백기선님이 진행하시는 스터디의 출석 체크 프로그램이다.   
> 백깃허브 레포지토리에 각 주차 이슈가 있고, 그 이슈에 코멘트로 링크를 올리면 출석이 인정된다.  
> [스터디 링크](https://github.com/whiteship/live-study)

<br>

## 구현 기능 목록
- 스터디 참석자 목록을 만든다.
- 스터디 참석자의 출석률을 계산하고 보여준다. 

<br>

## 필요한 객체
- 깃허브 (API)
- 깃허브 레포지토리 (API)
- 깃허브 이슈 (API)
- 참석자 목록
- 참석자 객체
